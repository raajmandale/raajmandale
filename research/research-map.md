
# ERANEST Systems Research Map

Founder: **Raaj Mandale**  
Organization: **ERANEST Technoware Pvt Ltd**

## Overview

The ERANEST research map represents a collection of system-level technologies focused on building **deterministic, survivable, and secure digital infrastructure**.

The work spans multiple domains including:

- Deterministic data architectures
- Fragment-based storage systems
- Survivable data infrastructure
- Pattern-based compute runtime
- AI risk intelligence systems
- Secure decision architectures

The goal is to replace fragile digital systems with **resilient, verifiable infrastructure**.

# Core Research Layers

## 1. Compute Infrastructure

### QBAIX

Experimental compute platform exploring next-generation infrastructure combining:

• High-performance compute  
• Deterministic execution models  
• Scalable runtime systems  

QBAIX acts as the long-term compute platform for deterministic systems.

## 2. Runtime Systems

### M‑OS Runtime

Pattern-based runtime architecture exploring deterministic compute behavior.

Key ideas:

• Pattern State Transition Graph (PSTG)  
• Deterministic scheduling  
• Scalable compute abstraction  
• Runtime adapters for heterogeneous hardware

Repository:
https://github.com/raajmandale/mos-runtime

---

## 3. Data Architecture

### Deterministic Fragment Graph (DFG)

DFG introduces a new model where **data becomes a recoverable graph instead of fragile files**.

Core idea:

```
File → Fragment → Graph → Reconstruction
```

Fragments are stored across multiple locations while metadata relationships allow deterministic reconstruction.

Repository:
https://github.com/raajmandale/dfg-demo-lab

---

## 4. Data Survivability Systems

### Digital Lifeline

Digital Lifeline ensures that information remains recoverable even when:

• Devices fail  
• Systems are destroyed  
• Storage becomes corrupted  

Architecture principles:

• Fragment-based storage  
• Multi-location survivability  
• Metadata-driven reconstruction  
• Zero full-file server storage

Repository:
https://github.com/raajmandale/digital-lifeline
