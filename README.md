# 🔬 Deterministic Fragment Graph (DFG)

A new architecture where **data becomes a recoverable graph instead of fragile files**.

Core flow:


File
↓
Fragment
↓
Graph
↓
Deterministic Reconstruction


DFG powers multiple systems:

• XPADI storage engine  
• XRebuild reconstruction engine  
• Digital Lifeline survivability system  

Repository:

https://github.com/raajmandale/dfg-demo-lab

• lost  
• destroyed  
• corrupted  

Architecture principles:

• fragment-based storage  
• multi-location survivability  
• metadata-driven reconstruction  
• zero full file storage on servers
